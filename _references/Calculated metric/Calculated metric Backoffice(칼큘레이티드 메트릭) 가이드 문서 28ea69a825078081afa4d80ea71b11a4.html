How to Create Calculated Metrics

Fill in the fields

  • Organization ID: Enter the Organization ID of the organization that you want to create the Calculated Metric for.
  • Key: Enter the identifier for the Calculated Metric that should be unique to the organization.
  • Display Name: Enter the display name of the Calculated Metric that will be shown in the Actuals Report and Trend Report.
    • The display name will appear in the metric option drop-down list.
    • The tooltip for the Calculated Metric will contain the configured formula.

Create formula

Rules

  • The formula must be in JSON format.
  • The formula can contain metrics, basic arithmetic operations (+ - * /), and real numbers.
  • Declare 'arithmetic', specify the basic arithmetic operation you want to use, and then enter the operands under 'fields'. (Input the left operand of the expression first, followed by the right operand.)
  • Since calculations are performed in the order they are grouped within the field, be sure to carefully consider the order of the expression and use parentheses when necessary.
  • You can use the field value from column G (metric_key) of the service specification metric sheet, or you can use the metric names that appear when copying and pasting the set values from the report.â€
  • Custom Metrics, Cohort Metrics or Self-serve Metrics can be used when creating formulas. However, to avoid errors in the reports, the corresponding metrics must actually exist. Therefore, it is advised to select these metrics and load the data, then copy the set values from the Actuals Report to confirm the metric names.
  • Note that existing Calculated Metrics cannot be used when creating formulas. If needed, copy the formula of the existing Calculated Metric and use it for the new formula.
